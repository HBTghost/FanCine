## Quick Start

  Install Node & Yarn (latest verson), then enter in command line:

```bash
$ git clone https://github.com/HBTghost/FanCine.git && cd FanCine
```

  Install dependencies:

```bash
$ yarn
```

  Start the server:

```bash
$ yarn serve
```

  App will opened automaticly at: http://localhost:6969
